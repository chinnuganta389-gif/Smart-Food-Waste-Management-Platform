# FoodShare - Smart Food Waste Management Platform

FoodShare is a web-based platform developed to reduce food waste and improve food accessibility by connecting food donors with individuals and organizations in need.

 Features

# Admin Module

* Dashboard Analytics
* State & City Management
* Donor Management
* Food Listing Management
* Request Monitoring
* Reports Generation

Donor Module

* Add Food Listings
* Manage Donations
* Track Requests
* View Impact Metrics

 # User Module

* Browse Available Food
* Request Food
* Track Request Status
* Contact Support

# Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB
* Authentication: JWT

## Objective

To minimize food wastage and create a sustainable food-sharing ecosystem through technology.
